# 项目进度

本文件只记录当前状态。技术参数见 [`tech.md`](tech.md)，现场记录见 [`grbl.md`](grbl.md) 和 [`magnet.md`](magnet.md)，旧方案见 [`archive/`](archive/)。

## 当前状态

- 硬件主形态已定：单层透明面板、引磁片棋片、外接 MOSFET 驱动电磁铁。
- XY 运动、PWM 控制、限位触发和基础串口控制已打通。
- 连续物理路径规划、普通走子、吃子流程和吃子区槽位管理已接入 CLI/API。
- 主仓库视觉链路已解耦，只接收外部视觉结果；`GhostVision` 负责采集、标定和识别。
- 视觉结果可转换为 `BoardState` / FEN，并接入 `Pikafish` 生成走法。

## 已验证

- 单层面板下基础拖动、L 型通道和擦边经过可用。
- 标准开局 15 步场景复验通过，包含视觉校对。
- 吃子专项 `C_capture_edge` 通过，包含被吃子入槽和进攻方后续移动。
- 演示触发约定已验证：GRBL Y 限位双击确认，三击仅重置软件位置。

## 待完成

- 实机完整对局演示联调与稳定性验收。
