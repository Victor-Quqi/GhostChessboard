# 幽灵棋盘技术规划

## 概述

中国象棋自动对弈棋盘，棋子通过底部磁力驱动在棋盘表面滑动。

## 当前方案

- 棋盘顶板采用 `2mm` 单层透明板，棋盘层贴在上表面
- 当前不采用永磁体锁定，不再继续更换更强的同类吸盘式电磁铁
- 电磁铁最低点不再增加额外厚度，优先直接压缩磁距
- 当前控制方式采用“按格执行”的单步基础动作，不走整条路径持续吸附拖动
- 旧方案和已放弃路线统一归档到 [`docs/archive/abandoned-options.md`](archive/abandoned-options.md)

## 系统结构

```
USB 摄像头（1080P，俯拍）
        │
        ▼
Intel NUC7i5BNH（Linux / Ubuntu）
  ├─ 视觉识别
  ├─ 象棋 AI
  └─ USB 串口 → G-code
        │
        ▼
ESP32 GRBL + TMC2209 ×2
  ├─ XY 运动
  └─ PWM → MOSFET → 电磁铁
```

## 机械与棋盘

- 顶板：`2mm` 透明亚克力或 PC
- 棋盘图案：现成 `PP/PVC` 棋盘层贴于上表面
- 棋子：底部贴 `Φ20×0.3mm` 引磁片
- 电磁铁：当前已验证 `5W` / `8W` 吸盘式型号；两者都能驱动，但都不是最终长期方向
- 木板固定：当前做法为在桐木板上打通孔后，使用 `4` 颗带垫圈的螺钉穿过木板，并以约 `8cm` 的间距拧入铝型材槽内的 `T` 型螺母，从而把木板固定到 XY 框架
- 侧面按钮优先，避免上表面入镜和误触

## 运动与控制

### 电磁铁驱动

- GRBL 激光接口输出的是约 `0V/5V` PWM 逻辑电平
- 电磁铁通过外接 MOSFET 低边开关驱动，不直接挂在板载接口上
- 指令约定：
  - `M3 Sxxx`：电磁铁通电
  - `M5`：电磁铁断电
  - `$30`：PWM 上限，当前按 `1000` 使用

### 坐标与格距

- `y` 轴平行于楚河汉界
- `x` 轴垂直于楚河汉界
- 当前控制格距：
  - `x = 41.2mm/格`
  - `y = 42.0mm/格`

### 单步动作

- 当前默认单步动作：
  - 空载前导 `pre`
  - `M3 Sxxx` 吸合
  - 带棋移动主段
  - `M5` 断电
  - 电磁铁空载回正 `overshoot`
- 当前默认初值：
  - `overshoot ≈ 16mm`
  - `pre ≈ 15mm`
- 当前方向是“逐格执行”，而不是整条路径持续吸着拖

### 路径规划

- 路径节点按棋盘点位中心处理
- 仅在空点通行
- 采用 4 邻接 BFS
- 普通走子按 BFS 路径逐格执行上述单步动作
- 吃子流程仍按“先移走被吃子，再移动进攻方”推进

## 棋盘与 XY 平台尺寸

- 平台：雕刻机框架，工作面积 `440mm × 505mm`
- 主棋盘：`9 × 10`
- 吃子区：右侧 `2 × 10`
- 点阵总数：`11 × 10`
- 棋盘纸标称点阵间距：`P = 41mm`
- 尺寸估算：
  - 长边方向：`10P + 2M = 440mm`
  - 短边方向：`9P + 2M = 399mm`

## 视觉识别与补光

- 摄像头：USB UVC，1080P，手动对焦
- 安装高度：约 `50cm` 俯拍
- 取流建议：USB2.0 下优先 MJPEG，尽量锁定曝光和白平衡
- 补光：`12V 2m 4000K` 正发光硅胶灯带

## 供电策略

- 前期优先使用各设备原装电源，降低联调复杂度
- 外设母线按 `12V` 设计，供步进电机、灯带、电磁铁使用
- 当前外设电源：`LRS-350-12`
- 控制板与 `12V` 外设母线必须共地

## 当前待继续验证的事项

- 继续细化 `x+ / x- / y+ / y-` 四个方向的补偿值
- 在更高棋局密度下复验擦边经过和误吸情况
- 把 BFS 路径执行正式接入完整走子与吃子流程
