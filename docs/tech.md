# 幽灵棋盘技术规划

## 概述

中国象棋自动对弈棋盘，棋子通过底部磁力驱动在棋盘表面滑动。

## 防误吸方案：永磁体锁定

**问题**：电磁铁拖动棋子经过其他棋子时，可能把其他棋子也吸走。

**方案**：每个棋位下方固定一个小永磁体。

- 永磁体把棋子"钉"在原位
- 电磁铁正下方时，吸力 > 永磁体，能拖走棋子
- 电磁铁侧向经过时，斜向吸力 < 永磁体锁定力，不会误吸


### 电磁铁选型与误吸风险（量级估算，待实测）

注：以下为初步估算，用于指导电磁铁选型；必须用小样（3×3 或 1×N）实测确认。

- 暂定型号（验证用）：GLX-P30/22（12V；约 5W；D=30mm；杯式/吸盘式电磁铁）。
- 方法 1（距离估算）：按 2mm 顶板 + 2mm 永磁体 + 3mm 隔离板 + 0.5~2mm 机械余量估算，电磁铁到引磁片的等效间隙约 7.5~9mm。结论：选型应关注“在 8~10mm 间隙仍能吸住”的能力，商家标注的 0 间隙吸力参考意义有限。
- 方法 2（实测锁定力）：做单棋位治具，测棋子从永磁体上被垂直拔起的临界力/重量（锁定力）。结论：电磁铁在实际间隙下的拉力建议至少达到锁定力的 1.5~2 倍（给装配偏差与拖动动态留裕量）。
- 材料影响：亚克力/PC 基本不屏蔽磁场；隔离板的主要作用是“机械隔离/定距”，避免电磁铁与永磁体阵列直接吸附/碰撞，同时让间隙可控。
- 误吸评估（数量级）：用”吸力随距离快速衰减”的近似估算，在点阵间距 P=41mm 下，相邻格距电磁铁远大于垂直间隙（几 mm），相邻格吸力远小于目标格（万分之一量级），直接误吸风险较低；更需要关注电磁铁被永磁体阵列吸引导致的阻力/卡滞与拖动摩擦。

### 电磁铁驱动

控制板（ESP32 GRBL 六轴）激光接口已有板载 MOSFET（XH2.54 三针：12V / GND / PWM），计划直接驱动电磁铁，无需外接驱动电路。实测 M3/M5 控制验证待完成。

### 电磁铁控制接口（初版已定）

目标：让“吸合/保持/断开”与 XY 运动严格同步，优先保证演示稳定性与实现成本。

- 控制路径：NUC（上位机，Linux）通过 USB 串口向 ESP32 GRBL 发送 G-code；GRBL 负责排队执行运动与外设控制。
- 输出引脚：使用控制板的“激光/主轴 PWM”输出端子（GND/5V/PWM/M3）。初版只接 **PWM→MOSFET Gate**（不做额外硬使能）。
- 指令约定：
  - `M3 Sxxx`：电磁铁通电，`S` 对应 PWM 占空比（吸力）。
  - `M5`：电磁铁断电。
  - 备注：GRBL 的 `S` 上限由 `$30` 决定（例如 `$30=1000` 时，`S1000` 代表满功率）。
- 吸力策略（软件层两档：吸合/保持，靠实测定参）：
  - 吸合：短时间用大 `S`（例如 100% 占空比 50~200ms）确保抓住棋子。
  - 保持/拖动：降到较小 `S` 持续拖动，降低发热与误吸概率。

## 视觉识别与补光（大部分确定）

- 环境：演示环境光照充足；可联网（AI 可走 API 或本地，后续再定）。
- 摄像头：USB UVC 普通摄像头模组，1080P，手动对焦；建议固定安装高度约 50cm，覆盖约 50cm 棋盘。
- 取流建议：USB2.0 下优先 MJPEG；软件侧固定帧率并尽量锁定自动曝光/白平衡，保证识别稳定。
- 补光：12V 2m 4000K 正发光硅胶灯带（10×10，围棋盘一圈/四边）。
- 供电建议：灯带接入 12V 主电源；如需调亮度可后续加 12V 旋钮调光器。
- 采购记录：详见 docs/bom.md。

## 棋盘与 XY 平台尺寸（初步建议）

约束：优先采购现成套件，减少定制加工与装配调试成本；以“演示可跑通”为第一优先级。

- 平台：雕刻机框架（工作面积 440mm×505mm，组件清单见下）。
- 点阵间距：P = 41mm（匹配已购棋盘纸）。
- 预留余量：M = 15mm/边（在不牺牲太多稳定性的前提下，为 440mm 短边留更多余量）。
- 主棋盘点阵：9 列 × 10 行（象棋标准点位）。
- 右侧吃子停放区：2 列 × 10 行（20 位，满足一盘棋吃子收纳）。
- 总点阵：11 列 × 10 行。
- 尺寸需求估算（用于匹配平台工作面积，需实测校准）：
  - 长边方向（建议放 505mm）：X_need = 10P + 2M = 440mm
  - 短边方向（建议放 440mm）：Y_need = 9P + 2M = 399mm

### 棋盘结构（初版方向）

- 外观方向：顶板覆盖整机外框；前期先能支撑与对位即可，后续再加侧围挡/外壳做精细化遮蔽。
- 上面板：2mm 透明亚克力/PC；棋盘图案建议背贴（打印贴膜/背胶），上表面保持光滑耐磨。
- 下层磁铁承载板：3mm 亚克力；永磁体直接胶粘定位（不挖孔）；与上板之间用隔柱/垫片留间隙，便于后续通过“距离”微调磁力；同时该板位于永磁体阵列与移动电磁铁之间，主要起机械隔离/定距作用，避免电磁铁与永磁体直接吸附/碰撞。
- 棋位永磁体（初版）：钕铁硼 Φ8×2mm（统一极性朝上）。
- 棋子引磁片（初版）：Φ20×0.3mm 圆片（贴于棋子底部）。
- 试验策略：先做 3×3 或 1×N 小样验证“锁定力/拖动能力/邻格干扰”，再铺全盘。
- 交互：按钮优先放在侧面，上表面尽量保持“纯棋盘面”，减少误触与入镜干扰。

## 供电策略（阶段性）

- 前期：优先使用各设备原装电源（尤其主机/NUC 一类），降低踩坑与调试成本。
- 运动/执行侧：12V 作为“外设母线”（步进电机、灯带、电磁铁等）。
- 外设母线电源：LRS-350-12（12V 29A，350W）。
- 信号侧：运动控制板与 12V 外设母线需要共地；电磁铁加续流保护，避免干扰控制与视觉。
- 后期：如果需要“单电源”演示，再评估统一 12V + DC-DC（给主机等）方案与功率余量。

## 拖动与吃子运动策略（初步推演，待验证）

前提：棋子贴面拖动；以下为算法方向的初步推演，需实机验证与调参。

- 路径规划：可先用点位中心为节点；仅在空点通行；采用 4 邻接（上/下/左/右）；用 BFS 求 from->to 路径。
- 普通走子：沿 BFS 路径逐点拖动到 to（转角减速等细节需实机验证）。
- 吃子：可先将 to 上被吃棋子拖到吃子区下一个空位，再将进攻方棋子从 from 拖到 to（具体时序可能要配合视觉检测做去抖）。
- 炮吃子：物理上不能跳过，可按 BFS 绕行到 to（棋规约束起终点，轨迹允许绕行）。
- 演示兜底：演示用棋谱/脚本可先离线仿真验证可达性；必要时再加入临时挪子让路策略（优先利用吃子区作为临时停车位）。
