# 幽灵棋盘技术规划

## 概述

中国象棋自动对弈棋盘，棋子通过底部磁力驱动在棋盘表面滑动。

## 防误吸方案：永磁体锁定（V1 暂不采用）

**问题**：电磁铁拖动棋子经过其他棋子时，可能把其他棋子也吸走。

**原方案（加分项，非现阶段硬要求）**：每个棋位下方固定一个小永磁体。

- 永磁体把棋子"钉"在原位
- 电磁铁正下方时，吸力 > 永磁体，能拖走棋子
- 电磁铁侧向经过时，斜向吸力 < 永磁体锁定力，不会误吸

注：2026-03-20 与 2026-03-25 的实测表明，当前 5W / 8W 吸盘式电磁铁在间隙增加后磁力裕量不足，且大吸合面带来明显边缘吸附与启动滞后。V1 演示版暂不采用永磁体锁定，否则只会进一步增加间隙与锁定力。


### 电磁铁选型与误吸风险（量级估算，待实测）

注：以下为初步估算，用于指导电磁铁选型；必须用小样（3×3 或 1×N）实测确认。

- 首轮验证型号：GLX-P30/22（12V；约 5W；D=30mm；杯式/吸盘式电磁铁）。
- 下一轮替换型号：商品页标称 12V / 8W / 40kg，外径约 40mm，高度约 20mm，吸合面直径约 18mm，安装孔 M5×12。
- 2026-03-25 实测结论：新 8W 吸盘式电磁铁在满功率下，单层板条件仅较 5W 略稳，双层板仍基本不可用；棋子更容易吸到吸合面边缘而非中心，拖动滞后与吸合面半径同量级。
- 方法 1（旧双层方案的距离估算）：按 2mm 顶板 + 2mm 永磁体 + 3mm 隔离板 + 0.5~2mm 机械余量估算，电磁铁到引磁片的等效间隙约 7.5~9mm。结论：选型应关注“在 8~10mm 间隙仍能吸住”的能力，商家标注的 0 间隙吸力参考意义有限。
- 方法 2（实测锁定力）：做单棋位治具，测棋子从永磁体上被垂直拔起的临界力/重量（锁定力）。结论：电磁铁在实际间隙下的拉力建议至少达到锁定力的 1.5~2 倍（给装配偏差与拖动动态留裕量）。
- 材料影响：亚克力/PC 基本不屏蔽磁场；隔离板的主要作用是“机械隔离/定距”，避免电磁铁与永磁体阵列直接吸附/碰撞，同时让间隙可控。
- 误吸评估（数量级）：用”吸力随距离快速衰减”的近似估算，在点阵间距 P=41mm 下，相邻格距电磁铁远大于垂直间隙（几 mm），相邻格吸力远小于目标格（万分之一量级），直接误吸风险较低；更需要关注电磁铁被永磁体阵列吸引导致的阻力/卡滞与拖动摩擦。
- 当前判断：V1 不再继续加码同类吸盘式电磁铁功率；若单层方案仍不足，下一步应更换磁铁类型或磁路结构，而不是继续追同类商品页参数。

### 电磁铁驱动

控制板（ESP32 GRBL 六轴）激光接口已实测可输出 `M3/M5` 对应的 PWM 控制信号，但当前测得为约 `0V/5V` 逻辑电平，不能直接作为 12V 电磁铁的功率输出。后续改为“GRBL PWM 信号 + 外接 MOSFET 低边开关”方案。

### 电磁铁控制接口（初版已定）

目标：让“吸合/保持/断开”与 XY 运动严格同步，优先保证演示稳定性与实现成本。

- 控制路径：NUC（上位机，Linux）通过 USB 串口向 ESP32 GRBL 发送 G-code；GRBL 负责排队执行运动与外设控制。
- 输出引脚：使用控制板的“激光/主轴 PWM”输出端子（GND/5V/PWM/M3）。初版按 **PWM→MOSFET Gate** 使用，不直接挂接电磁铁线圈。
- 指令约定：
  - `M3 Sxxx`：电磁铁通电，`S` 对应 PWM 占空比（吸力）。
  - `M5`：电磁铁断电。
  - 备注：GRBL 的 `S` 上限由 `$30` 决定（例如 `$30=1000` 时，`S1000` 代表满功率）。
- 吸力策略（软件层两档：吸合/保持，靠实测定参）：
  - 吸合：短时间用大 `S`（例如 100% 占空比 50~200ms）确保抓住棋子。
  - 保持/拖动：降到较小 `S` 持续拖动，降低发热与误吸概率。

### 当前结构方向（2026-03-25 更新）

- V1 结构定为“3mm 单层透明板 + 上贴 PP/PVC 棋盘层 + 无永磁体锁定”。
- 为尽量压缩磁距，允许电磁铁组件与面板下表面轻微受控滑擦；接触面优先使用 PTFE/UHMW/特氟龙脚贴等低摩擦、可更换耗材，避免金属硬磨面板。
- 当前不再继续更换更强的同类吸盘式电磁铁；如单层方案仍不足，后续应转向新的磁路结构。
- 面板若仅四周固定，需要关注中部下垂；必要时增加隐藏支撑或限高点，避免中心区域磁距继续变大。
- 当前面板支撑方案为“前后两块桐木侧板托板 + 上方少量点胶防窜位（非主承重）”；侧板按 `400×250×15mm` 采购，并固定在 XY 框架前后侧的型材上。
- 侧板与运动机构的避让以实装为准；当前俯视估算滑车走到四角时基本可避开侧板，若后续发现极限位置轻微干涉，可通过加厚限位块小幅牺牲行程换取避让。

## 棋盘与 XY 平台尺寸（初步建议）

约束：优先采购现成套件，减少定制加工与装配调试成本；以“演示可跑通”为第一优先级。

- 平台：雕刻机框架（工作面积 440mm×505mm，组件清单见下）。
- 点阵间距：P = 41mm（匹配已购棋盘纸）。
- 预留余量：M = 15mm/边（在不牺牲太多稳定性的前提下，为 440mm 短边留更多余量）。
- 主棋盘点阵：9 列 × 10 行（象棋标准点位）。
- 右侧吃子停放区：2 列 × 10 行（20 位，满足一盘棋吃子收纳）。
- 总点阵：11 列 × 10 行。
- 尺寸需求估算（用于匹配平台工作面积，需实测校准）：
  - 长边方向（建议放 505mm）：X_need = 10P + 2M = 440mm
  - 短边方向（建议放 440mm）：Y_need = 9P + 2M = 399mm

### 棋盘结构（初版方向）

- 外观方向：顶板覆盖整机外框；前期先能支撑与对位即可，后续再加侧围挡/外壳做精细化遮蔽。
- 上面板：V1 现方案为 3mm 透明亚克力/PC。
- 棋盘图案：使用现成 PP/PVC 棋盘层贴于上表面，优先保证最小磁距与后续可更换性。
- 面板支撑：当前优先采用前后两块桐木侧板从下方托住面板，而非让热熔胶承担主结构受力；热熔胶仅作为边缘点固、防窜位的临时措施。
- 面板下表面允许与电磁铁组件轻微滑擦；建议在电磁铁最低点贴 PTFE 鼠标脚贴或同类低摩擦耐磨片，作为可更换滑脚。
- V1 不再设置下层磁铁承载板与棋位永磁体，整机先按单层面板验证拖动稳定性。
- 棋子引磁片：暂继续使用较大尺寸圆片（当前为 Φ20×0.3mm 方案）；仅靠缩小引磁片不能解决滞后，且已观察到单层板拖动能力下降。
- 试验策略：优先验证“3mm 单层板 + 上贴棋盘层 + 受控轻擦”下的连续拖动、启动滞后与邻格干扰，再决定是否需要隐藏支撑或更换磁路。
- 装配余量：侧板上表面应尽量共面；后续如发现面板中部仍下垂，可优先补隐藏限高点、机械压片或局部支撑，而不是增加面板层数。
- 交互：按钮优先放在侧面，上表面尽量保持“纯棋盘面”，减少误触与入镜干扰。

## 供电策略（阶段性）

- 前期：优先使用各设备原装电源（尤其主机/NUC 一类），降低踩坑与调试成本。
- 运动/执行侧：12V 作为“外设母线”（步进电机、灯带、电磁铁等）。
- 外设母线电源：LRS-350-12（12V 29A，350W）。
- 信号侧：运动控制板与 12V 外设母线需要共地；电磁铁加续流保护，避免干扰控制与视觉。
- 后期：如果需要“单电源”演示，再评估统一 12V + DC-DC（给主机等）方案与功率余量。

## 拖动与吃子运动策略（初步推演，待验证）

前提：棋子贴面拖动；以下为算法方向的初步推演，需实机验证与调参。

- 路径规划：可先用点位中心为节点；仅在空点通行；采用 4 邻接（上/下/左/右）；用 BFS 求 from->to 路径。
- 普通走子：沿 BFS 路径逐点拖动到 to（转角减速等细节需实机验证）。
- 吃子：可先将 to 上被吃棋子拖到吃子区下一个空位，再将进攻方棋子从 from 拖到 to（具体时序可能要配合视觉检测做去抖）。
- 炮吃子：物理上不能跳过，可按 BFS 绕行到 to（棋规约束起终点，轨迹允许绕行）。
- 演示兜底：演示用棋谱/脚本可先离线仿真验证可达性；必要时再加入临时挪子让路策略（优先利用吃子区作为临时停车位）。
