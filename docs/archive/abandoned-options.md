# 已放弃方案归档

本文件只保留历史记录，不代表当前方案。当前有效方案以 [`docs/tech.md`](../tech.md) 为准。

## 双层板 + 永磁体锁定

曾考虑在每个棋位下方增加永磁体，用来降低误吸风险。后续放弃，原因如下：

- 会额外增加磁距，直接压缩电磁铁可用吸力
- 还会提高起步脱离所需的锁定力，进一步放大拖动难度
- 2026-03-20 与 2026-03-25 的实测表明，当前 `5W / 8W` 吸盘式电磁铁在更大间隙下已没有足够裕量

## 继续追同类更强吸盘式电磁铁

曾尝试从约 `5W` 吸盘式电磁铁升级到商品页标称 `12V / 8W / 40kg` 的同类型号。后续不再继续沿这条路线推进，原因如下：

- 单层板下仅略有改善，不是数量级提升
- 双层板下仍基本不可用
- 更大的吸合面更容易形成环状吸力峰值，把棋子拉向吸合面边缘，落点偏置更明显
- 现阶段问题更像磁路形式不匹配，而不是单纯功率不够

## 3mm 单层透明板

`3mm` 单层透明板曾作为 V1 候选方案。2026-04-05 的现场验证后，不再作为当前默认方案，原因如下：

- 在当前装配状态下，`3mm` 板基本不可用
- `2mm` 单层透明板已能稳定完成基础拖动
- 继续保留 `3mm` 板采购记录，但不再把它写成当前方案

## 缩小棋子引磁片

曾尝试把棋子底部引磁片裁小，想减少边缘吸附和落点偏置。后续放弃，原因如下：

- 单层板下拖动能力进一步下降
- 不能从根本上解决吸盘式电磁铁的环状吸力峰值问题
