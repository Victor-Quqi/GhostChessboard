# 电磁铁驱动与磁力验证

## 驱动方案

控制板（ESP32 GRBL 六轴）上有板载 MOSFET 驱动的**激光接口**（XH2.54 三针：12V / GND / PWM），可直接驱动电磁铁，无需外接驱动电路。

接线：
- 激光接口 `12V` → 电磁铁正极
- 激光接口 `GND` → 电磁铁负极
- `PWM` 脚不接（GRBL 内部控制）

G-code 控制：
- `M3 S1000` — 满占空比，吸合阶段用
- `M3 S500` — 50% 占空比，拖动阶段降功率减发热
- `M5` — 断电

板上其他相邻端子说明（无需使用）：
- `5V`：控制板 5V 输出，限流小，不能驱动电磁铁
- `PWM(0~10V)`：模拟主轴调速信号，给变频器用，非功率输出
- `M8` / `M3`：冷却液控制 IO，3.3V/5V 电平，非功率输出
