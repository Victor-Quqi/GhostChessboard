# GRBL 调试记录

## 硬件连接

- **控制板**：ESP32 GRBL 六轴，CH340 USB 串口
- **串口**：COM18（CH340），波特率 115200
- **驱动模块**：TMC2209 ×2，插 X、Y 座；方向与 A4988 **相反**（芯片面朝 ESP32 侧，板上标注"2209 则相反，反插会烧"）
- **限位开关**：XY 平台原装 3 线 XH2.54，只接 COM→GND、NC→限位端子，NO 线不接

## GRBL 参数

> **重要**：CH340 开串口会触发 ESP32 复位，`$` 参数不会持久化，每次连接后需重新发送。

### 已验证（必须每次连接后发送）

| 参数 | 值 | 说明 |
|------|-----|------|
| `$3` | `2` | 方向反转掩码：反转 Y 轴（bit1） |
| `$5` | `3` | 限位引脚反转：X+Y（NC 接法）。注：写入返回 ok 但读回显示 0，原因未明，运行时实际生效 |

### 出厂默认（待标定）

| 参数 | 值 | 说明 |
|------|-----|------|
| `$30` | `1000` | 主轴/PWM 最大值（电磁铁占空比上限） |
| `$100` | `640` | X 步进/mm |
| `$101` | `640` | Y 步进/mm |
| `$110` | `24000` | X 最大速率 mm/min |
| `$111` | `24000` | Y 最大速率 mm/min |
| `$120` | `50` | X 加速度 mm/s² |
| `$121` | `50` | Y 加速度 mm/s² |
