# GhostChessboard 幽灵棋盘

中国象棋自动对弈棋盘 —— **嵌入式**课程项目

## 项目简介

棋子底部贴引磁片，由棋盘下方 XY 平台上的电磁铁拖动，在棋盘表面自动滑动，可用于人机对弈或 AI 演示。

## 设计原则

- **硬件从简**：团队软件/算法能力较强，硬件偏弱，优先选用成熟套件
- **演示优先**：人力、时间、资金有限，以"能跑通演示"为首要目标
- **嵌入式约束**：课程要求，需全嵌入式实现
